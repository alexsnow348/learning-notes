Kedro, Kedro-Viz
----
- ML/Data Science project တွေအတွက် သေသေချာချာစနစ်တကြ organised ဖြစ်အောင်နဲ့ reproduce လုပ်တာ လွယ်အောင်လို့ လုပ်ပေးတဲ့ python framework တစ်ခုဖြစ်ပါတယ်။

### Installation

```bash
pip install kedro
```

### Useful command

```bash
kedro new # to create new kedro project
```
