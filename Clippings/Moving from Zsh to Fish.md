## Moving from Zsh to Fish
---
### Installation 

1. Install fish

```bash
brew install fish
fish --version
# fish, version 3.1.2
```

2. Install oh-my-fish aka omf

```bash
$ curl -L https://get.oh-my.fish | fish
$ omf version
# Oh My Fish version 7
```

3. Install fzf.fish, Example of using FZF via Ctrl-r in fish.

```bash
$ fisher install PatrickF1/fzf.fish
```
