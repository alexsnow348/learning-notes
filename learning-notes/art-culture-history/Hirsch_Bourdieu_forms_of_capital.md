# Hirsch vs. Bourdieu
```pgsql
                          ┌─────────────────────────────┐
                          │     NEOLIBERAL SYSTEM       │
                          │  (Markets, Competition,     │
                          │   Meritocracy Narrative)    │
                          └──────────────┬──────────────┘
                                         │
                                         ▼
                         ┌──────────────────────────────────┐
                         │     INTENSIFIED COMPETITION      │
                         └──────────────┬───────────────────┘
                                         │
                                         ▼
   ┌─────────────────────────────────────────────────────────────────────────┐
   │                HIRSCH: POSITIONAL GOODS (scarce by nature)              │
   │    ─────────────────────────────────────────────────────────────────     │
   │    • Elite schools     • High-status jobs     • Desirable housing        │
   │    • Prestigious clubs • Exclusive networks   • Symbolic status           │
   └──────────────┬───────────────────────────────────────────────────────────┘
                  │ These goods cannot expand with growth (zero-sum)
                  ▼
           ┌─────────────────────┐
           │  SOCIAL BOTTLENECK  │
           └─────────────────────┘
                  │
                  ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│          BOURDIEU: FORMS OF CAPITAL USED TO WIN COMPETITION                  │
│  ─────────────────────────────────────────────────────────────────────────     │
│   ECONOMIC CAPITAL: money, property, ability to pay for advantages             │
│   CULTURAL CAPITAL: language, tastes, credentials, knowledge                  │
│   SOCIAL CAPITAL: networks, connections, ties to institutions                 │
│   SYMBOLIC CAPITAL: prestige, reputation, legitimacy                          │
└──────────────┬──────────────────────────────────────────────────────────────┘
               │
               │  Elites apply these capitals to gain exclusive access
               ▼
        ┌──────────────────────────┐
        │  CONTROL OF POSITIONAL   │
        │          GOODS           │
        └───────────┬──────────────┘
                    │
                    ▼
      ┌──────────────────────────────────────────────────┐
      │  REPRODUCTION OF ADVANTAGE (Bourdieu)            │
      │  - Elites pass capital to their children         │
      │  - Institutions reward existing capital          │
      │  - Meritocracy disguises structural inequality   │
      └──────────────────────────────────────────────────┘
                    │
                    ▼
       ┌─────────────────────────────────────────────┐
       │ SOCIAL LIMITS TO GROWTH (Hirsch)            │
       │ - More growth → more competition            │
       │ - No increase in actual access              │
       │ - Rising frustration, inequality, anxiety   │
       └─────────────────────────────────────────────┘

```
